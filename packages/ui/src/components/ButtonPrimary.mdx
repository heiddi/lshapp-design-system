# ButtonPrimary

The `ButtonPrimary` component is a versatile button that can be used throughout the application for primary actions.

## Usage

```tsx
import { ButtonPrimary } from 'ui';

function MyComponent() {
  return (
    <ButtonPrimary 
      label="Press Me" 
      onPress={() => console.log('Button pressed')} 
    />
  );
}
```

## Accessibility

The ButtonPrimary component includes the following accessibility features:

- Has `accessibilityRole="button"` to convey the correct semantics
- Uses the `label` prop as the `accessibilityLabel` for screen readers
- Provides appropriate visual feedback for different states (normal, pressed, disabled)
